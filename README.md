## Features Breakdown

### Calorie Calculator
- Calculates Basal Metabolic Rate (BMR) using Mifflin-St Jeor equation
- Factors in activity level to determine Total Daily Energy Expenditure (TDEE)
- Provides maintenance calories and fat loss deficit (500 kcal reduction)
- User-friendly form with validation

### Responsive Design
- Mobile-first approach
- Hamburger menu for mobile devices
- Flexible grid layouts that adapt to screen size
- Touch-friendly buttons and inputs

### Design Elements
- Purple (#6C63FF) and dark grey (#2A2A2A) color scheme
- Smooth scroll behavior
- Hover effects and transitions
- Modern card-based layout

## Customization

### Colors
Edit the CSS variables in the `<style>` section:
```css
:root {
    --primary: #6C63FF;
    --primary-dark: #5a52d5;
    --secondary: #2A2A2A;
    --accent: #00D2D3;
}
```

### Content
- Update text content directly in the HTML
- Modify calculator logic in the `<script>` section
- Adjust styling in the embedded `<style>` section
