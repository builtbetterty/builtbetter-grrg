# Quick Upload Instructions

## Option 1: Install Xcode Tools & Run Script (Recommended)

1. **Install Xcode Command Line Tools:**
   ```bash
   xcode-select --install
   ```
   (A popup will appear - click "Install")

2. **After installation, run:**
   ```bash
   ./upload_to_github.sh
   ```

## Option 2: Use GitHub Desktop (Easiest - No Terminal Needed)

1. Download GitHub Desktop: https://desktop.github.com/
2. Sign in with your GitHub account
3. File â†’ Add Local Repository
4. Select this folder: `/Users/temp/Desktop/webiste`
5. Click "Publish repository"
6. Repository name: `builtbetter-`
7. Make sure the owner is: `builtbetterty`
8. Click "Publish repository"

## Option 3: Manual Upload via GitHub Website

1. Go to: https://github.com/builtbetterty/builtbetter-
2. Click "uploading an existing file"
3. Drag and drop all files from this folder
4. Commit changes

## Enable GitHub Pages (After Upload)

1. Go to: https://github.com/builtbetterty/builtbetter-/settings/pages
2. Under "Source", select **main** branch
3. Click **Save**
4. Your site will be live at: `https://builtbetterty.github.io/builtbetter-/`
